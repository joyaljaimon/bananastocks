# 🍌 BananaStocks

> **Understanding the stock market shouldn't require a finance degree. Sometimes, all you need is a monkey and a banana farm.**

BananaStocks is an AI-powered web application that explains stocks, investing concepts, and financial news using one simple analogy:

**A monkey running a banana business. 🍌🐒**

Instead of drowning users in financial jargon, BananaStocks translates complex investing concepts into entertaining, easy-to-understand stories. Whether it's earnings reports, P/E ratios, market crashes, or dividends, everything becomes part of the monkey's banana empire.

---

## 🎯 Why I Built This

I've always been fascinated by the stock market.

The more I learned about investing, the more I realized that many people avoid it—not because it's uninteresting, but because it's intimidating. Financial language often feels unnecessarily complicated, making beginners think investing is only for experts.

As someone who enjoys both software engineering and simplifying difficult ideas, I wanted to build something that makes learning about stocks genuinely fun.

So I asked myself:

> **"What if Wall Street was just a banana farm?"**

That simple question became BananaStocks.

---

## 🍌 Example

Instead of saying:

> "Apple's revenue increased 12% year-over-year, improving operating margins and earnings per share."

BananaStocks explains it like this:

> **Monkey Joe owns a banana farm. Last year he sold 100 bananas. This year he sold 112 bananas while spending almost the same amount feeding his workers. Since he earned more bananas without spending much extra, every monkey who owns a piece of the farm gets a bigger slice of the profits. Investors love that. 🍌**

Same concept.

Much easier.

Much more memorable.

## 💡 Vision

The goal isn't just to explain stocks.

It's to make finance feel approachable.

Too many educational platforms overwhelm beginners with terminology before helping them understand the *why* behind investing.

BananaStocks takes the opposite approach:

> **Teach the idea first. Teach the vocabulary second.**

Because once someone understands the story, the financial terms become much easier to learn.

## 🖥️ Running Locally

Clone the repository:

```bash
git clone https://github.com/joyaljaimon/bananastocks.git
```

Navigate into the project:

```bash
cd bananastocks
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
